# 로또 미션 - 기능 목록

## 1. 도메인 모델

### 1.1 Lotto 클래스
- 1.1.1 로또 번호 6 개를 저장하는 Lotto 클래스 생성
- 1.1.2 로또 번호 유효성 검증 (6 개, 중복 없음, 1~45 범위)
- 1.1.3 로또 번호 오름차순 정렬 기능 구현
- 1.1.4 Lotto 클래스 toString() 메서드 구현 ([1, 2, 3, 4, 5, 6] 형식)

### 1.2 Prize Enum
- 1.2.1 Prize Enum 생성 (1 등~5 등)
- 1.2.2 Prize Enum 에 당첨 금액 정의 (2B, 30M, 1.5M, 50K, 5K 원)
- 1.2.3 Prize Enum 에 일치 개수 정의 (6, 5+보너스, 5, 4, 3)
- 1.2.4 Prize Enum 에 등수 표시 문자열 정의

### 1.3 MatchResult Enum
- 1.3.1 MatchResult Enum 생성 (3 개~6 개 일치, 5 개+보너스)
- 1.3.2 MatchResult 에 당첨 금액 조회 기능 추가

## 2. 도메인 서비스

### 2.1 LottoGenerator
- 2.1.1 Randoms.pickUniqueNumbersInRange() 를 사용한 로또 자동 생성
- 2.1.2 1~45 범위에서 중복 없는 6 개 숫자 추출
- 2.1.3 생성된 로또 번호 리스트 반환

### 2.2 LottoTicketGenerator
- 2.2.1 구입 금액 입력 받기
- 2.2.2 구입 금액으로 구매 가능한 로또 수량 계산 (1000 원 단위)
- 2.2.3 로또 수량만큼 LottoTicket 리스트 생성
- 2.2.4 구매 금액 유효성 검증 (1000 원 단위, 양수)

### 2.3 PrizeCalculator
- 2.3.1 당첨 번호와 보너스 번호 저장
- 2.3.2 로또 번호와 당첨 번호 비교
- 2.3.3 일치 개수 계산 (보너스 번호 포함 여부 확인)
- 2.3.4 MatchResult 도출
- 2.3.5 당첨 금액 계산

### 2.4 PrizeStatistics
- 2.4.1 당첨 결과별 카운터 초기화
- 2.4.2 모든 로또의 당첨 결과 집계
- 2.4.3 각 등수별 당첨 개수 반환
- 2.4.4 당첨 통계 문자열 생성 (3 개~6 개 일치 순)

### 2.5 ROICalculator
- 2.5.1 총 구입 금액 저장
- 2.5.2 총 당첨 금액 계산
- 2.5.3 수익률 계산 ((총당첨금액 - 총구입금액) / 총구입금액 * 100)
- 2.5.4 수익률 소수점 둘째 자리 반올림
- 2.5.5 수익률 문자열 생성 (ex. 62.5%)
